# 🚀 Интеграция с SaleBot - Инструкция

## Как это работает

### 1. **Пользователь приходит из Telegram бота**
Когда пользователь переходит на квиз из SaleBot, он может попасть по ссылке с параметрами:

```
https://ваш-домен.ru?user_id=12345&chat_id=67890
```

или

```
https://ваш-домен.ru?client_id=12345
```

### 2. **Система сохраняет все параметры**
При загрузке главной страницы (Home) система автоматически:
- Считывает все параметры из URL
- Сохраняет их в localStorage
- Логирует в консоль браузера (для отладки)

### 3. **Пользователь проходит квиз**
Система собирает ответы:
- Вес, рост, возраст
- Цель по весу
- Уровень активности
- Сроки достижения цели

### 4. **На странице Paywall кнопка отправляет в Telegram**
При клике на кнопку "Получить план" формируется ссылка:

```
https://t.me/easymeclub_bot?start=plan_USERID_вес75_цель65_минус10_8нед_15мая
```

**Параметры в ссылке:**
- `USERID` - user_id или client_id от SaleBot (или "unknown" если не передан)
- `вес75` - текущий вес пользователя
- `цель65` - целевой вес
- `минус10` - разница в кг
- `8нед` - количество недель до цели
- `15мая` - дата достижения цели

## 📝 Настройка SaleBot

### В вашем боте создайте обработчик команды `/start`

Когда пользователь вернётся из квиза, вы получите параметр типа:
```
plan_123456_вес75_цель65_минус10_8нед_15мая
```

### Пример парсинга в боте:

```python
# Пример на Python
start_param = message.text.replace('/start ', '')
# start_param = "plan_123456_вес75_цель65_минус10_8нед_15мая"

parts = start_param.split('_')
# parts = ["plan", "123456", "вес75", "цель65", "минус10", "8нед", "15мая"]

user_id = parts[1]  # "123456"
current_weight = parts[2].replace('вес', '')  # "75"
target_weight = parts[3].replace('цель', '')  # "65"
difference = parts[4].replace('минус', '')  # "10"
weeks = parts[5].replace('нед', '')  # "8"
target_date = parts[6]  # "15мая"

# Теперь можете:
# 1. Записать данные в метки пользователя
# 2. Отправить персональное сообщение
# 3. Предложить оплату со скидкой
```

## 🔧 Технические детали

### Файлы, которые были изменены:

1. **`/src/app/utils/urlParams.ts`** - утилиты для работы с параметрами
   - `saveUrlParams()` - сохраняет параметры из URL
   - `getSavedParams()` - получает сохранённые параметры
   - `buildTelegramLink()` - формирует ссылку на бота

2. **`/src/app/pages/Home.tsx`** - добавлен захват параметров при входе

3. **`/src/app/pages/Paywall.tsx`** - добавлен обработчик кнопки с редиректом

### Как изменить username бота

Если нужно изменить имя бота с `easymeclub_bot` на другое:

Откройте `/src/app/pages/Paywall.tsx` и найдите строку:
```typescript
const telegramLink = buildTelegramLink(quizData, 'easymeclub_bot');
```

Замените `'easymeclub_bot'` на нужный username.

## 🐛 Отладка

### Где смотреть логи?

1. Откройте консоль браузера (F12)
2. При загрузке главной страницы с параметрами увидите:
   ```
   ✅ SaleBot параметры сохранены: {user_id: "12345", chat_id: "67890"}
   ```

3. При клике на кнопку в Paywall увидите:
   ```
   🚀 Редирект в Telegram: https://t.me/easymeclub_bot?start=plan_12345_...
   ```

### Проверка в localStorage

В консоли браузера выполните:
```javascript
localStorage.getItem('saleBotParams')
```

Должно вернуть JSON с параметрами от SaleBot.
