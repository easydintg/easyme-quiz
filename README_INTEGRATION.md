# 🎯 EasyME Quiz Funnel - Интеграция с SaleBot

## Быстрый старт

### Что сделано:

✅ **Захват параметров от SaleBot**
- При входе на сайт с параметрами `?user_id=123` они автоматически сохраняются
- Работает с любыми параметрами: `user_id`, `client_id`, `chat_id` и др.

✅ **Передача данных обратно в бот**
- При клике на кнопку "Получить план" пользователь перенаправляется в Telegram
- Все данные квиза передаются в параметре `/start`

---

## 📋 Формат данных

### Вход (от SaleBot):
```
https://ваш-домен.ru?user_id=12345&chat_id=67890
```

### Выход (в Telegram бота):
```
https://t.me/easymeclub_bot?start=plan_12345_вес75_цель65_минус10_8нед_15мая
```

**Параметры:**
- `12345` - user_id пользователя
- `вес75` - текущий вес (75 кг)
- `цель65` - целевой вес (65 кг)
- `минус10` - нужно сбросить (10 кг)
- `8нед` - за сколько недель (8)
- `15мая` - дата достижения цели

---

## 🧪 Тестирование

### 1. Откройте сайт с параметрами:
```
http://localhost:5173?user_id=test123
```

### 2. Откройте консоль браузера (F12)
Вы должны увидеть:
```
✅ SaleBot параметры сохранены: {user_id: "test123"}
```

### 3. Пройдите квиз до конца

### 4. На странице Paywall нажмите кнопку
В консоли вы увидите:
```
🚀 Редирект в Telegram: https://t.me/easymeclub_bot?start=plan_test123_...
```

## 🛠 Настройка

Если нужно изменить username бота, откройте:
`/src/app/pages/Paywall.tsx`

Найдите строку (около строки 57):
```typescript
const telegramLink = buildTelegramLink(quizData, 'easymeclub_bot');
```

Замените `'easymeclub_bot'` на ваш username бота.
